# ЕГЭ-База. №4 Преобразования выражений

### Задания 

1.  Площадь треугольника $S$ со сторонами $a, b, c$ можно найти по формуле Герона $S = \sqrt{p(p - a)(p - b)(p - c)}$, где $p = \frac{a + b + c}{2}$.  Найдите площадь треугольника, если длины его сторон равны 4, 51, 53.

2.  Перевести температуру из шкалы Фаренгейта в шкалу Цельсия позволяет формула $t_c = \frac{5}{9}(t_r - 32)$, где $t_c$ — температура в градусах по шкале Цельсия, $t_r$ — температура в градусах по шкале Фаренгейта.  Скольким градусам по шкале Цельсия соответствует 257 градусов по шкале Фаренгейта?

3.  Количество теплоты (в джоулях), полученное однородным телом при нагревании, вычисляется по формуле $Q = cm(t_2 - t_1)$, где $c$ — удельная теплоёмкость $\left(\frac{Дж}{кг \cdot К}\right)$, $m$ — масса тела (в кг), $t_1$ — начальная температура тела (в кельвинах), $t_2$ — конечная температура тела (в кельвинах).  Пользуясь этой формулой, найдите $Q$, если $t_2 = 409 \, К$, $c = 450 \, \frac{Дж}{кг \cdot К}$, $m = 4 \, кг$ и $t_1 = 405 \, К$.

4.  Мощность постоянного тока (в ваттах) вычисляется по формуле $P = I^2R$, где $I$ — сила тока (в амперах), $R$ — сопротивление (в омах).  Пользуясь этой формулой, найдите $R$ (в омах), если $P = 180 \, Вт$ и $I = 6 \, А$.

5.  Потенциальная энергия тела (в джоулях) в поле тяготения Земли вблизи её поверхности вычисляется по формуле $E = mgh$, где $m$ — масса тела (в килограммах), $g$ — ускорение свободного падения (в м/с$^2$), $h$ — высота (в метрах), на которой находится это тело, относительно поверхности.  Пользуясь этой формулой, найдите $m$ (в килограммах), если $g = 9,8 \, м/с^2$, $h = 0,5 \, м$, а $E = 49 \, Дж$.

6. Теорему синусов можно записать в виде $\frac{\sin \alpha}{a} = \frac{\sin \beta}{b}$, где $a$ и $b$ — две стороны треугольника, $\alpha$ и $\beta$ — углы треугольника, лежащие против них соответственно.  Пользуясь этой формулой, найдите величину $\sin \alpha$, если $a = 27, b = 20$, $\sin \beta = \frac{2}{3}$.

7.  Энергия заряженного конденсатора $W$ (в Дж) вычисляется по формуле $W = \frac{q^2}{2C}$, где $C$ — ёмкость конденсатора (в Ф), $q$ — заряд на одной обкладке конденсатора (в Кл).  Найдите энергию (в Дж) конденсатора ёмкостью $10^{-4} \, Ф$, если заряд на его обкладке равен 0,0012 Кл.

8. Если $p_1, p_2$ и $p_3$ — простые числа, то сумма всех делителей числа $p_1 \cdot p_2 \cdot p_3$ равна $(p_1 + 1)(p_2 + 1)(p_3 + 1)$.  Найдите сумму всех делителей числа 114.

9.  Площадь треугольника можно вычислить по формуле $S = \frac{abc}{4R}$, где $a, b, c$ — стороны треугольника, а $R$ — радиус окружности, описанной около этого треугольника.  Пользуясь этой формулой, найдите площадь $S$, если $a = 10, b = 9, c = 17$ и $R = \frac{85}{8}$.

10. Зная длину своего шага, человек может приближённо подсчитать пройденное им расстояние $s$ по формуле $s = nl$, где $n$ — число шагов, $l$ — длина шага.  Какое расстояние прошёл человек, если $l = 70 \, см$, $n = 1800$? Ответ выразите в метрах.

11. Среднее геометрическое трёх чисел $a, b$ и $c$ вычисляется по формуле $s = \sqrt[3]{abc}$.  Вычислите среднее геометрическое чисел 2, 4, 27.

12. Площадь треугольника можно вычислить по формуле $S = \frac{abc}{4R}$, где $a, b, c$ — стороны треугольника, а $R$ — радиус окружности, описанной около этого треугольника.  Пользуясь этой формулой, найдите площадь $S$, если $a = 3, b = 25, c = 26$ и $R = \frac{325}{24}$.
